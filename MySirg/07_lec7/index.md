
==================
primitive types
==================

# javaScript has eight basic primitive data types and one non primitive

1. String  ---  "Abhi"
2. Number  ---  23,43.3
3. Boolean --  true,false
4. Null    ---  null
5. Undefined  == undefined 
6. Symbol   == Symbol('id')
7. Bigint  == 123n




=========================
Non primitive datatypes
=========================

1. Object 
ex - {name:"abhi,age:12};
==> Key value pairs

2. Array
ex - [12,32,33]
==> ordered list of value

3. Function
ex - function(){
    // code
}
==> reusable code
