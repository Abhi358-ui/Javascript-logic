## Window object

- Window ek global object ha matlab ek object ha jo hame browser create karke hmare document ke sath attach kr deta ha 

- Aur is window object ke paas bahut sara property rahta ha function rahta ha jo hmare kaam ko bahut easy kr dete ha 

- Javascript me console.log() bhi usi global object ki ek property ha 

- Ye window object browser hmare liye create karta ha 

- hmara pura html content window object ki ek property hota ha document usme store rahta ha object ki form me 

- Hmara js html ke sath kuch bhi edit nhi kr sakti isliye html file ko pahle el object me convert karte ha uske baad us object me hm manipulation karte ha 

- Isi object ek tree ke format me store rahta ha to use document object model kahte ha 

- Javascript me sb kuch ek object ki tarah store hota ha key value pair me kyuki js me object me manipulation karna bahur easy hota ha aur dom bhi aise hi hota ha


- Window object global object hota ha isliye wo sb jagah available rahta ha aur hm use bina naam ke bhi access kr lete ha 





## Document methods

- document.innerHTML us tag ke andar jitne bhi html styles aur jitne bhi ha sb kuch batata ha including styles 

- document.innerText bs jo content ui pr show ho raha ha use hi dikhata ha

- document.innerTextContent bs html ko chhor ke jo sb dikh raha ha wo sb dikhata ha 

- document.querySelector() hota ha jiska use hota ha js me css ke tarike se kisi bhi tag ko select karna 

- document.querySelectorAll() isme us tag se related sabhi tag ek html collection ki tarah aa jate ha
